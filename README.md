# directed-slop

An LLM-powered content platform where articles are generated by AI, guided by human prompts. The human provides the topic and direction; the LLM writes the full article as a self-contained, interactive HTML page.

No build step. No frameworks. Each article is a single HTML file with embedded CSS, JavaScript, fonts (via Google Fonts), and CDN libraries. Black and white design, Noto Naskh Arabic as the base font.

Content is written in Arabic with technical terms kept in English.

## Writing a New Article

Use the following prompt template with an LLM. ChatGPT 5.4 is recommended — it can follow online links directly, letting you reference papers or articles as source material.

```
Write a blog article following the instructions in @instructions.md

- **Topic**: [REPLACE WITH TOPIC]
```

You can also point the model to an online source:

```
Write a blog article following the instructions in @instructions.md

- **Topic**: Intelligent AI Delegation (from the paper: https://arxiv.org/pdf/2602.11865)
```

The LLM will generate a self-contained HTML file in `./content/` and update `./index.html` at the project root to link to it.

The [instructions.md](instructions.md) file contains the full writing guidelines, design principles, and content rules.

## Project Structure

```
index.html               # Landing page listing all articles
content/
├── article-1.html       # Self-contained article page
├── article-2.html       # Self-contained article page
└── ...
instructions.md          # Full article generation guidelines
```
